# Example Queries

Try asking an agent questions like:

- What is the difference between [[concepts/raw-sources|Raw Sources]] and [[concepts/compiled-wiki|Compiled Wiki]]?
- How does [[concepts/compounding-knowledge|Compounding Knowledge]] differ from basic retrieval?
- What should happen during [[workflow/ingest|Ingest]]?
- How would [[workflow/lint|Lint]] find missing pages or stale synthesis?
- What belongs in the [[concepts/schema|Schema]]?

A good answer can be saved back into the wiki through [[workflow/query|Query]].
